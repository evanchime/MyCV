WhatsApp AI Chatbot (Python version) | March 2024

Python, Git, GitHub, Flask, WhatsApp Business Platform API, AWS EC2, Llama 2 Large Language Model, LangChain, ngrok

An AI Chatbot written in python. The user sends a message from their WhatsApp Business app. The WhatsApp Business Platform receives and forwards the message to a webhook. The webhook calls a web app, running in an AWS EC2 instance, passing along the user message. The web app calls Meta's Llama 2 LLM with the user message. Llama 2 returns the answer. The web app sends the answer and user phone number to the WhatsApp Platform. The WhatsApp platform sends the answer to the app

The major achievement and challenge that was overcame was getting the chatbot to remember previous conversations and not allowing the chatbot and its conversations to live forever
